Cara Pakai

Setup dalam
hitungan menit

1
Step
Deploy ke Vercel
Fork repo, connect ke Vercel, set environment variables (Pterodactyl domain + API key admin). Done.
2
Step
Setup GitHub DB
Buat repo baru di GitHub, isi GH_TOKEN di env vars. Data user & server otomatis tersimpan di sana.
3
Step
Set Kredensial Admin
Edit config.js — isi username, email, dan password admin. Login otomatis terdeteksi sebagai admin.
4
Step
Buat User & Server
Login sebagai admin → buat user (free/premium) → user login → buat server sendiri. Admin punya full kontrol.
